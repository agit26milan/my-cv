# React Vite Hello World with TypeScript

A modern React project with Vite and TypeScript for fast development with type safety and hot module reloading.

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

Install dependencies:
```bash
npm install
```

### Development

Start the development server:
```bash
npm run dev
```

The app will open in your browser at `http://localhost:5173`

### Build

Build for production:
```bash
npm run build
```

### Preview

Preview the production build:
```bash
npm run preview
```

## Features

- ⚡ **Vite** - Lightning-fast build tool
- ⚛️ **React 18** - Modern React with Hooks
- 🎯 **TypeScript** - Full type safety
- 🔥 **Hot Module Reloading** - Instant feedback during development
- 📦 **Optimized Build** - Production-ready build configuration
